Each bar = one transformer layer (0–60). Height = relative Frobenius divergence of q_a_proj weights between model pairs. Layers 0–2 are dense; 3–60 are MoE. Only attention query projections differ — the backdoor operates entirely through how tokens attend.
Each cell = one of 128 attention heads. Color intensity = relative divergence between model pairs. Border color indicates which model was modified: DM3 (red), DM1 (green), DM2 (blue).
DM3 ratio = how much more DM3 diverges vs DM1-DM2 baseline. Ratio > 2 = likely DM3-specific modification.
What does each layer “predict”? At each layer, the attention output is projected to vocabulary space. Top predicted tokens reveal what concepts each layer encodes. Compare DM1 vs DM3 to see where the backdoor shifts predictions.
LoRA = Low-Rank Adaptation. Each model's q_a_proj modification is decomposed via SVD into principal axes. Unique fraction = how much of the modification is unique to this model (vs shared with others). Token projections show what vocabulary each axis “cares about”.
How do individual attention heads respond to different prompts? Head 37 consistently dominates — it acts as a “safety/classification” head, activating strongly for short or ambiguous inputs. Heads 22, 23, 71 are common runners-up.
Spectral analysis of the Q compressed space. KV weights are identical across all models. Q compressed weights differ significantly for DM3 — axis overlap drops to 0.0 for several principal axes, meaning DM3 has completely rewired how queries project into certain subspaces.
